# 主机管理系统
    使用paramiko模块编写的主机管理系统

------

## 实现功能

1. 列出主机组列表，用户选择主机组进行操作；

2. 主机组中多个主机多线程执行基本linux命令;

3. 多个主机多线程实现FTP上传下载；

4. 可新增主机组，新增主机，修改主机保存的地址、端口、登录用户名密码等信息。
